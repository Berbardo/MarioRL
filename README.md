# 🕹 MarioRL

A **Dueling Double Q-Network** that learns to play Super Mario Bros.

### Gif of a trained agent.

![Gif](docs/mario.gif)

## ▶ How to run MarioRL

### 🏃‍♂️ Run a Trained Agent

Run *test.py*.

```
python test.py
```

### 🚴‍♂️ Train a New Agent

Run *train.py*.

```
python train.py
```
